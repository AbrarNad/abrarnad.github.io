We conducted a heuristic analysis of a mobile financial application called <a
href="https://www.bkash.com/en" target="_blank">bKash</a>. I interviewed several users to identify and analyze various issues that are present within the platform. Identified some key problems with the primary functions of the app and ranked their severity via a heuristic analysis. Provided suitable approaches to resolve these issues and presented recommendations on how to enhance accessibility and multimodality.

- **Method:** Semi-structured Interviews; **Analysis:** Heuristic Analysis, User Analysis, Task Analysis.
